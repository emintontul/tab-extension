# CloseMyTabs

A Chrome extension to batch close tabs by domain.

## Features

- **Group tabs by domain** - See all open tabs organized by website
- **Most tabs first** - Sites with the most tabs appear at the top
- **One-click close** - Close all tabs from a site with one button
- **Window filter** - Switch between current window or all windows
- **Pinned tabs protected** - Pinned tabs are never shown or closed
- **15 languages** - Auto-detects browser language

## Supported Languages

English, Türkçe, Deutsch, Français, Español, Italiano, Português, Русский, 日本語, 한국어, 中文, العربية, Nederlands, Polski, Svenska

### Manual Install
1. Download or clone this repository
2. Go to `chrome://extensions`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the extension folder
